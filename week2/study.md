# 📘 Today I Learned

### 1. 오늘 배운 내용

- HTML과 CSS만으로 정적 웹페이지 만들기
- media query로 화면 크기에 따라 레이아웃 바꾸기

### 2. 핵심 정리 (내 언어로)

- grid-template-columns로 몇 열로 나눌지 정할 수 있음
- position: absolute는 부모가 relative일 때 그 안에서 위치를 잡음
- @media (max-width: 768px) 이런 식으로 화면 크기별로 다른 스타일 적용 가능!
- CSS 파일을 따로 만들어서 link 태그로 HTML에 연결해야 함
