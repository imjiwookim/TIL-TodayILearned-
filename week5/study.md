# 📘 Today I Learned

### 1. 오늘 배운 내용

- Vite + React 프로젝트 생성 및 개발 서버 실행 방법
- UI를 컴포넌트 단위로 분리하는 방법
- props를 통해 데이터를 전달하는 방법
- JSX 문법 규칙

### 2. 핵심 정리 (내 언어로)

- 컴포넌트는 UI를 역할별로 나눈 조각으로, 재사용이 가능하다
- props는 부모가 자식 컴포넌트에 데이터를 전달하는 방법이다
- JSX는 HTML처럼 생겼지만 class 대신 className을 써야 한다
- 데이터는 App에서 시작해 하위 컴포넌트로 흘러간다
