# 📘 Today I Learned

### 1. 오늘 배운 내용

- JavaScript React 프로젝트에 TypeScript를 적용하는 마이그레이션 방법
- interface와 type으로 컴포넌트 props와 데이터 구조를 정의하는 방법
- useState에 제네릭으로 상태 타입을 명시하는 방법
- 이벤트 핸들러에 ChangeEvent, KeyboardEvent 등 적절한 타입을 지정하는 방법
- 공통 타입을 별도 파일(src/types/lion.ts)로 분리해 재사용하는 방법

### 2. 핵심 정리 (내 언어로)

- TypeScript는 코드를 실행하기 전에 타입 오류를 미리 잡아줘서 런타임 에러를 줄여준다
- interface로 props 타입을 정의하면 잘못된 값을 전달했을 때 편집기에서 바로 오류가 표시된다
- useState<Lion[]>처럼 제네릭을 쓰면 배열 안에 어떤 데이터가 들어오는지 명확해진다
- 공통 타입을 한 파일에 모아두면 여러 컴포넌트에서 같은 타입을 일관되게 사용할 수 있다
