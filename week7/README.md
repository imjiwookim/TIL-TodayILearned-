# 📘 Today I Learned

### 1. 오늘 배운 내용

- React Router를 사용해 목록 페이지와 상세 페이지로 화면을 분리하는 방법
- URL 경로 파라미터(/lions/:id)로 특정 데이터를 식별해 화면에 표시하는 방법
- useSearchParams로 필터/정렬/검색 옵션을 URL 쿼리 파라미터와 연동하는 방법
- 여러 페이지가 데이터를 공유할 때 상태를 최상위 컴포넌트(App)에서 관리하는 방법

### 2. 핵심 정리 (내 언어로)

- React Router를 쓰면 URL은 바뀌지만 페이지 전체가 새로 로드되지 않는 SPA 방식으로 동작한다
- URL에 id를 넣으면(/lions/3) 그 숫자로 데이터를 찾아서 해당 사람의 상세 페이지를 보여줄 수 있다
- 필터/검색 조건을 URL에 저장하면 새로고침해도 같은 화면이 유지되고 링크로 공유도 가능하다
- 목록과 상세 페이지가 같은 데이터를 써야 하므로 App에서 상태를 관리하고 props로 내려줬다
